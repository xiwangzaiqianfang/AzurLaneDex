# 碧蓝航线图鉴 (AzurLane Dex)

一个为《碧蓝航线》玩家设计的舰船收集管理工具，使用 **WinUI 3** 和 **C#** 制作，部分代码使用 AI 工具辅助开发。


## ✨ 主要功能

- **舰船图鉴**：完整的舰船数据库（编号、名称、阵营、稀有度、改造、特殊兵装等）
- **收集进度跟踪**：标记已获得、突破次数、120级、誓约、改造、特殊兵装
- **批量操作**：快速标记多艘舰船的状态
- **筛选与搜索**：按舰种、阵营、稀有度、属性加成等多维度筛选，支持船名/和谐名/活动名搜索
- **阵营科技点**：实时计算每个阵营的科技点总和（获得 + 满破 + 120级），进度条展示
- **全局属性加成**：统计全舰队（或指定舰种）的耐久、炮击、雷击等加成总值
- **多账户系统**：支持创建多个用户，各自独立保存进度，可设置管理员账户
- **数据备份/恢复**：导出/导入账户数据，创建本地备份点
- **旧版数据迁移**：可从原 Python 版的 `ships_static.json` 自动转换成新版格式
- **应用更新**：支持 MSIX 打包，通过 Windows 应用安装程序（AppInstaller）实现自动更新

## 🖥️ 技术栈

- **框架**: WinUI 3 (Windows App SDK 1.x)
- **语言**: C# 10 / .NET 6
- **数据格式**: JSON 本地存储
- **UI 模式**: MVVM (INotifyPropertyChanged)
- **打包方式**: MSIX (支持旁加载)

## 📦 安装与更新

### 直接安装 MSIX 包
1. 前往 [Releases](https://github.com/xiwangzaiqianfang/AzurLaneDex/releases/latest) 下载最新的 `.msix` 文件。
2. 双击安装，需要系统开启**旁加载模式**（开发者设置中允许安装外部应用）。
3. 首次运行时会提示创建初始账户。

### 自动更新（推荐）
应用已集成 **Windows 应用安装程序**。安装后，点击「设置」→「更新」→「检查更新」，系统会自动检查新版本并下载安装（无需重复手动下载）。

## 🚀 从旧 Python 版迁移数据

如果你之前使用过 Python 版的 `ships_static.json`，可按以下步骤迁移：

1. 打开 WinUI 3 版本，进入「设置」→「数据迁移」。
2. 点击「导入旧版静态数据并转换」，选择旧版 JSON 文件。
3. 应用会自动转换为新格式并覆盖静态数据，请重启应用生效。

> ⚠️ 注意：旧版用户进度（`ships_state.json`）可直接放在新版本对应的账户目录下，无需转换。

## 🛠️ 开发者编译

### 环境要求
- Windows 11 或 Windows 10（版本 1809+）
- Visual Studio 2022（17.0+）并安装 **Windows 应用开发** 工作负载
- Windows App SDK 1.4 或更高

### 克隆与生成
```bash
git clone https://github.com/xiwangzaiqianfang/AzurLane-Dex-WinUI.git
cd AzurLane-Dex-WinUI
```

打开 AzurLaneDex.sln，在 Visual Studio 中编译并运行。
若要生成 MSIX 包，右键打包项目 → 发布 → 创建应用程序包。
