⚓ 碧蓝航线图鉴

AzurLane Dex · 为指挥官打造的舰船收集伴侣
WinUI 3 C# / .NET 6 MSIX 打包 多账户 自动更新
✨ 立即安装 ✨
通过 Windows 应用安装程序一键安装
📦 手动下载：GitHub Releases · 最新 MSIX 包

💡 若未能成功安装程序，请检查系统是否开启旁加载模式（设置 → 应用 → 高级应用设置）

📋 核心功能
📚

完整图鉴

编号、名称、阵营、稀有度、舰种、改造信息、特殊兵装等数据一览无余

✅

进度跟踪

标记已获得、突破次数、120级、誓约、改造、特殊兵装，实时统计收集率

⚙️

批量操作

一键标记多艘舰船的状态，支持全选/反选，大幅提升管理效率

🔍

智能筛选 & 搜索

按舰种、阵营、稀有度、属性加成筛选；支持船名/和谐名/活动名搜索

🏆

阵营科技点

自动计算每个阵营的科技点总和（获得+满破+120级），进度条一目了然

📊

全局属性加成

统计全舰队或指定舰种的耐久、炮击、雷击等加成总值

👥

多账户系统

支持多用户独立保存进度，可设置管理员账户，数据互不干扰

💾

备份 & 迁移

导出/导入账户数据，创建备份点，并支持从旧 Python 版数据自动转换

🔄

多源更新

集成更新功能，可手动检查下载更新，多源检查下载更省心

🛠️ 技术栈
WinUI 3 Windows App SDK 1.4+ C# 10 / .NET 6 MVVM JSON 本地存储 MSIX 旁加载

📦 安装方式

  1. 点击上方【安装 AzurLaneDex】按钮，系统将自动下载 .zip 文件，解压按照 Readme.txt 安装
  2. 首次运行会提示创建账户，按指引完成设置即可开始使用

🚀 从旧版（Python 版）迁移数据

  • 打开新版应用 → 「设置」→「数据迁移」→ 选择旧的 ships_static.json 文件
  • 程序将自动转换为新格式并覆盖静态数据，重启后生效
  • 用户进度文件 ships_state.json 可直接放入对应账户目录（无需转换）

🧪 开发者编译

环境：Visual Studio 2022 + Windows 应用开发工作负载。克隆仓库后打开 AzurLaneDex.sln，即可编译运行。
